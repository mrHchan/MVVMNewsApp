# MVVMNewsApp

# Overview
> Kotlin-based Android news app helps users stay up-to-date on the latest headlines.

# Tools and Tech stack used :
- Kotlin
- XML
- MVVM Architecture
- Room database library
- Android Studio
- Third party libraries like retrofit, Glide
